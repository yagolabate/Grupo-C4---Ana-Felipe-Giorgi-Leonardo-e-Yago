Sistema Mecânico

O projeto de Sistema Mecânico consistiu, como bem diz o nome, na modelagem de um sistema mecânico, sendo o termo “mecânico” no sentido utilizado em física e não no sentido de “oficinas mecânicas”. O modelo deveria responder alguma pergunta, podendo ser de tipo explicativa, preditiva ou referente a algum parâmetro do sistema. Diferente do projeto Dinâmica Populacional, os alunos tiveram liberdade para escolher o sistema que iriam modelar. No meu caso, eu modelei um salto de paraquedas.

Materiais utilizados

1 Computador.

Detalhes(inicio, desenvolvimento, conclusão)

Para começar a realização desse projeto, foi necessária uma pesquisa sobre saltos de paraquedas para compreender um pouco mais sobre o sistema, como o tipo de avião utilizado, o tipo de paraquedas, a altura dos saltos e o tempo médio de salto. Além disso, foram feitas pesquisas para determinar quais tipos de forças agiam no paraquedista durante o salto. Uma fez feitas as pesquisas, foram realizadas equações diferencias para prever o comportamento nas dimensões x e y do movimento do paraquedista.

A segunda etapa foi responder alguma pergunta referente ao modelo. A pergunta respondida pelo meu modelo foi: Para quais áreas de paraquedas, o paraquedista aterrissaria em menos de 300 segundos? A partir disso, foram feitas variações nas áreas dos paraquedas e observando o que ocorria com o tempo de queda. Como conclusão, foi determinado que para paraquedas com áreas menores que 24 metros quadrados, o paraquedista aterrissaria em menos de 300 segundos.
