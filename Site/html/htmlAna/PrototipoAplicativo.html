MusicBox

Primeiro passo - com o intuito de obter uma partitura, o usuário pode optar entre escolher um arquivo prè-existente de seu computador(clicando no botão 'escolher arquivo') ou realizar a gravação do aúdio(clicando no botão 'gravar'). Segundo passo - a partir do momento em que o usuário já decidiu o arquivo que irá utilizar para obter a respectiva partitura, a próxima etapa é confirmar o arquivo escolhido. Terceiro passo - confirmado o arquivo, o usuário deve clicar no botão 'gerar cromagrama', responsável por transferir as frequências de áudio para um cromagrama. Quarto passo - o usuário pode optar por gerar uma partitura a partir do cromagrama clicando no botão 'gerar partitura'. Quinto passo - o usuário pode visualizar a partitura gerada clicando no botão 'mostrar partitura'. Resumindo, existe uma ordem a ser seguida para que o programa funcione de forma correta: escolher ou gravar um arquivo de aúdio; gerar uma partitura a partir do mesmo; visualizar a partitura na interface.

Materiais utilizados

The PyAudio library was used to record computer microphone input. The Librosa library was used to compute a chromagram. The LilyPond project was used in order to plot the sheet music. The Tkinter framework was used to write the GUI.

This application receives a musical input from your computer microphone and generates its musical score as the output. All is very easy to do trough a GUI, with the whole program written in pure Python.
