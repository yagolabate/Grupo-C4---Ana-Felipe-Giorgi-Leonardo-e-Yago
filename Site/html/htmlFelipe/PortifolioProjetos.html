Portifólio de Projetos

O portifólio de projetos foi um site criado para mostrar todos os projetos realizados no curso de engenharia.

Materiais utilizados

► Computador

Detalhes(inicio, desenvolvimento, conclusão)

O portifólio de Projetos é um projeto de Co-Design de Aplicativos que se iniciou em setembro. O objetivo é criar um site em html que tenha a pagina dos projetos, a pagina individual de cada pessoae a pagina da engenharia em geral. Esse projeto acompanha uma wiki, na qual os integrantes do grupo colocam os dados de como vai ser o Portifólio.
