Projeto Final

Matéria:

Design de Software

Breve Descrição:

Este projeto foi realizado para a matéria de Design de Software, foi o último projeto do semestre dessa matéria. Foram formados grupos de três pessoas e o objetivo era criar um aplicativo que interagisse com o usuário.

Materiais utilizados

Não foi utilizado nenhum tipo de material, além do programa Python, o qual trabalhamos o semestre inteiro.

Detalhes(inicio, desenvolvimento, conclusão)

O Projeto Final de Design de Software foi o último projeto do semestre que se iniciou no meio de maio e teve duração de aproximadamente um mês. O objetivo do projeto era criar um aplicativo que interagisse com o usuário, podendo ser um jogo, uma rede social, um aplicativo de uso pessoal, etc. O aplicativo tinha que ser obrigatóriamente feito com codificação Python. Nos primeiros dias de trabalho, cada grupo desprendeu algumas ideias. Depois de escolhida a ideia principal, alguns sketches foram feitos das interfaces do programa. O restante do trabalho consistiu em codificação, até que no final, com o trabalho pronto e funcional, os projetos foram apresentados aos professores e aos alunos.
