Jogo da Velha

Este projeto consiste na contrução de um codigo de joguo da velha em python

Materiais utilizados

Computador.

Detalhes(inicio, desenvolvimento, conclusão)

Este projeto foi realizado por grupos de duas pessoas, neste projeto criamos um codigo do jogo da velha em que foi feito uma divisao entre tabuleiro e jogo, no qual um dos integrantes do grupo criava o codigo responsavel pelo tabuleiro do jogo da velha e o outro integrante devia criar o codigo responsavel pelo sistema do jogo, no final ambos os codigos deviam "conversar" entre si, sendo que essa interação devia ser impecavel para que o jogo no total funcionase de forma correta.
