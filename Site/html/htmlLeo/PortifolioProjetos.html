Portifolio dos projetos

Este projeto consiste na construção de um site que seja um portifolio de projetos dos alunos

Materiais utilizados

Computador

Detalhes(inicio, desenvolvimento, conclusão)

Este projeto foi realizado por grupos de 4 pessoas, no qual se deve criar uma pagina inicial que de acessoa ao portifolio de cada integrante do grupo, toda a construção do site deve ser justificada quanto ao alinhamento, espaçamento, padrão, posicionamento e cor, no meu grupo nos criamos nosso site inspiradados no estilo das paginas do insper tanto nas cores quanto no posicionamento.
