Projeto Final

Este projeto conciste na criação de um jogo ou aplicativo atraves do python utilizando a interface do tkinter ou do pygames

Materiais utilizados

Computador

Detalhes(inicio, desenvolvimento, conclusão)

Este projeto foi realizado por grupos de 3 pessoas, inicialmente se tem um BrandStorm para gerar opções de projetos os quais sao apresentados ao professor que indica os mais interressantes e plauseveis de se concretizar ate a data de entrega,tal projeto abordou todos os conceitos apreendidos durante o curso assim como classes, funções, dicionarios, listas, entre outros diversos conceitos, tambem tivemos que implementar nossa autonomia uma vez que fomos incentivados a buscar a reposta de como um alguns codigos funcionavam na internet, meu grupo escolheu fazer um tamagotchi com a tematica pokemon, neste jogo vc pode escolher entre os tres pokemons iniciais da primeira geração, ou seja charmander, bulbasaur e squirtle, ao decorrer do jogo vc deve pode alimentar, dar banho e colocar seu pokemon para durmir, tambem a um sistema de loja no qual vc pode comprar itens para realizar as ações anteriores como tambem comprar equipamentos como roupa chapeu e ate bigode para seu pokemon, o jogo tambem possui um sistema de evolução no qual ao atingir um certo nivel seu pokemon evolui.
