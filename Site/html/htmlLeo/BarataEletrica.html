Barata Elétrica

Este projeto consciste em desconstruir um brinquedo proposto afim de realizar uma analise de sua estrutura e de seus componentes,e em seguida criar uma "barata eletrica" reutilizando apenas o motor do brinquedo

Materiais utilizados

Computador

Motor

Bateria

Impressora 3D

Detalhes(inicio, desenvolvimento, conclusão)

Neste projeto em grupo com um colega desconstruimos inicialmente o brinquedo que recebemos, analisando todas suas partes atraves de skets estrumentados para entender melhor como tal objeto funcionava, como foram pensadas as medidas de cada parte, que juntas o formavam, após compreender toda sua estrutura começamos a idealizar nossa "barata eletrica" que seria composta por uma "casca" considerando as medidas mais adequadas para que o segundo componente, ou seja os componentes internos ficassem devidamente postos, para idealizar sua "casca" utilizamos o "Fusion", programa de sketch 3D,e em seguida para materializa-la realizamos a impressao deste sket atraves das impressora 3D, tendo assim todos os componentes necessarios para construir a "barata eletrica" passamos para a fase de prototipação, a unificação dos componentes foi feita araves de fita dupla face, afim de fixar os componentes internos, logo obtivemos o que queriamos, uma "barata eletrica" que andasse em linha reta.
